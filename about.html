Our Mission

We built the Barber Booking Agent to make appointment management painless for independent barbers and their clients. No bloated software, no heavy infrastructure — just a clean Python tool that runs anywhere Python does.

What We Do

The Barber Booking Agent is a lightweight command-line application that helps barbers manage their schedules, services, and clients. Bookings are stored locally in JSON, and notifications can be delivered via console output or email through any SMTP provider.

Why It's Different

  • Zero dependencies — uses only the Python standard library.
  • Portable — runs on any machine with Python 3.7 or newer.
  • Transparent — your data lives in a plain JSON file you control.
  • Open — easy to read, modify, and extend.
