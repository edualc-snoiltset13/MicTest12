Welcome

A simple Python CLI agent that manages barber appointment bookings with notifications for both the barber and the client.

Features

  • Register barbers with time slots, services, and pricing
  • Book, reschedule, and cancel appointments
  • View and search bookings by barber or client name
  • Console and email (SMTP) notifications
  • Local JSON persistence between sessions

Get Started

python barber_booking_agent.py
