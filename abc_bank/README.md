# abc_bank


---

### Project Description

This project is a **full-stack FinTech banking application** designed to simulate real-world financial systems. It features a high-performance **React Single-Page Application (SPA)** integrated with a secure and scalable **Spring Boot backend**.

The application supports core banking operations such as **deposit, withdrawal, and fund transfers**, built with a strong focus on **security, data integrity, and auditability**. It implements **JWT-based authentication**, role-based access control, and detailed logging to ensure secure and traceable user activity.

On the infrastructure side, the project follows modern **DevOps practices**, including **Docker-based containerization**, automated **CI/CD pipelines using GitHub Actions**, and **cloud integration with AWS (S3)** for storage and deployment. It also includes an **SMTP-based notification system** for transactional alerts and user communication.

Overall, this project demonstrates how to design, build, secure, and deploy a **production-ready, scalable full-stack application** using industry-standard tools and practices.
