About

This website is a minimal example of a Python web application using the Flask framework. It illustrates how a few small files can produce a fully working multi-page site.

Stack

  • Python 3 - the language
  • Flask - the web framework
  • Jinja2 - templating engine (bundled with Flask)
  • Plain CSS - styling, no build step required

Run it locally

pip install -r requirements.txt
python app.py

Then open http://localhost:5000 in your browser.
