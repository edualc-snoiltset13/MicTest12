About

This project demonstrates how to build a tab-based static website served by Python's built-in HTTP server.

It uses only the standard library, so there are no dependencies to install.

  • HTML for structure
  • CSS for styling and animations
  • Vanilla JavaScript for tab switching
  • Python http.server for serving
