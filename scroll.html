Scroll Down

A demo page with smooth scrolling, a sticky header, scroll progress, and a back-to-top button.

↓

Introduction

This page demonstrates a few common scroll patterns in a single file. As you scroll, watch the progress bar at the top fill in, and notice how the header stays pinned. Once you scroll past the hero, a back-to-top button appears in the bottom right.

Click any nav link to jump smoothly to that section. Everything works with no dependencies — just open the file in a browser.

Features

Sticky Header

Stays visible as you scroll.

Progress Bar

Tracks how far you've read.

Smooth Anchors

Nav links scroll instead of jump.

Back to Top

One-click return to the top.

Responsive Grid

Cards adapt to width.

Zero Deps

One file, no libraries.

Details

The scroll progress bar uses the document's scrollTop divided by the total scrollable height. It's recalculated on every scroll event and applied as a CSS width.

The sticky header uses position: sticky; top: 0 with a translucent background and a backdrop blur — a small touch that keeps text underneath readable as it scrolls past.

Smooth scrolling comes from html { scroll-behavior: smooth } — no JS needed for the anchor links.
